# realestatebychefj-app

This is a real estate client tools app for Chef J at realestatebychefj.com.

## Stack
- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS

## Brand colors
- Navy: `#0b1f3a`
- Gold: `#c9a052`
- Cream: `#f5f0e8`

## Notes
- All costs shown in the app are approximate estimates.
- Dev server runs on port 3000.
